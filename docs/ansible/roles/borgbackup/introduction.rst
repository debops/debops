Introduction
============

The ``debops.borgbackup`` Ansible role can be used to setup backup using
borgbackup and borgmatic.

Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.3.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops.borgbackup
